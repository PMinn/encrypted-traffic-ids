# 型別驗證

## 軟體
使用微軟的 Pylance 及 My Type Checker

## 安裝 mypy
```
pip install mypy
```

## 安裝型別輔助套件 (已包含於 requirements.txt)
[Python Type Stubs](https://github.com/microsoft/python-type-stubs/tree/main) 安裝：
```
pip install git+https://github.com/microsoft/python-type-stubs.git 
```

## 安裝 pre-commit 設定檔 (已包含於 requirements.txt)
```
pre-commit install
```
